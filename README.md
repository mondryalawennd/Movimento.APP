# Movimento.APP

Aplicação Angular para gerenciamento de movimentos manuais, integrada com API ASP.NET Core.

## Tecnologias utilizadas

- Angular 16
- TypeScript
- Bootstrap 5
- RxJS
- ASP.NET Core (API)
- PostgreSQL / SQL Server (API)
- Git / GitHub

## Pré-requisitos

- Node.js >= 18
- npm >= 9
- Angular CLI >= 16
- Git
